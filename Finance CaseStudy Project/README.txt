This is a finance project that mainly focuses on forecasting Tesla's deliverables, revenues, gross profit, operating expenses, WACC etc.  
The excel file consists of the following input sheets:
Drivers: Consists of basic information that would be used for all calculations throughout the case study 
P & L / Balance Sheet input: Consists of the Profit/Loss and Balance Sheet input for the years 2014, 2015, 2016, 2017
Workings: This sheet is followed by all the finding and forecasts of deliverables, revenues, gross profit, operating expenses, WACC etc
